Api for Tfood school project

About this project

This project was assigned to me and my colleague at school for subject RAO. This is a simple web application that will provide local businesses a platform to sell their products either B2B or B2C. This project aims to encourage customers to buy local products rather than buying the same products from supermarkets and connectivity between local businesses.

Api routes

  • api/company - access/modify company data
  • User - generate token, authorization and authentication
  • Order - access/modify orders depending on user role
  • Offers - modify orders if user has role of a company and view/order it if user is customer
